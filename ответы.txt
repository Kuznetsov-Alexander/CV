1) Изображения в OpenCV представляются в виде многомерных массивов NumPy:
Цветное изображение: 3D-массив с размерами (высота, ширина, каналы) в формате BGR.
Оттенки серого: 2D-массив (высота, ширина).

2)Принцип отображения окон в OpenCV
Окна создаются с помощью cv2.imshow(), который отображает массив как изображение.
Флаги создания окон:
- cv2.WINDOW_NORMAL: Окно с изменяемым размером.
- cv2.WINDOW_AUTOSIZE: Автоматический размер под изображение.
- cv2.WINDOW_FREERATIO: Без сохранения пропорций.

3) 3. Управление параметрами отображения окон
- cv2.namedWindow(): Создает окно с указанными свойствами.
- cv2.resizeWindow(): Изменяет размер окна.
- cv2.moveWindow(): Перемещает окно на экране.
- cv2.setWindowTitle(): Задает заголовок окна.

4) Класс для работы с видеопотоком
Класс: cv2.VideoCapture.

Основные методы:
-open(): Подключается к источнику.
-read(): Читает следующий кадр.
-release(): Освобождает ресурсы.
-isOpened(): Проверяет подключение.

5) Источники видеопотока
-Локальные устройства: Веб-камеры
-Файлы: Путь к видеофайлу
-Сетевые источники: IP-камеры HTTP-запросы (например, http://192.168.1.100:8080/video).

6) Принципы работы с изображениями в потоке
Цикл обработки:
while True:
    ret, frame = cap.read()
    if not ret:
        break
    # Обработка frame
Метод read():
Возвращает кортеж (ret, frame).
ret — флаг успешности чтения (True/False).
frame — текущий кадр в виде numpy-массива.

7) Что такое FourCC?
FourCC (Four Character Code) — 4-символьный код видеокодека.
Примеры: XVID, MJPG, H264.
Указывает кодек для кодирования видео при записи в cv2.VideoWriter

8) Класс VideoWriter
Назначение: Запись видео в файл.
Инициализация:
fourcc = cv2.VideoWriter_fourcc(*'XVID')
out = cv2.VideoWriter('output.avi', fourcc, 30.0, (640, 480))
Методы:
write(frame): Записывает кадр.
release(): Завершает запись.

9) Проверка cv2.waitKey(1) & 0xFF == 27
Код 27: Клавиша ESC.
0xFF: Маска для корректной обработки на 64-битных системах.
Проверка нажатия клавиши ESC для выхода из цикла.

10) Формат HSV
Компоненты:
Hue (Оттенок): Угол на цветовом круге (0–180 в OpenCV).
Saturation (Насыщенность): Интенсивность цвета (0–255).
Value (Яркость): Уровень яркости (0–255).
